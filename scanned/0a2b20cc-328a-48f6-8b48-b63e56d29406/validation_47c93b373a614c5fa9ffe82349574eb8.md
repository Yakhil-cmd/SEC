### Title
SNS Governance Allows `StakeMaturity` on Dissolved Neurons — Missing State Validation — (`File: rs/sns/governance/src/governance.rs`)

### Summary

The SNS Governance canister's `stake_maturity_of_neuron` function does not validate that the target neuron is in an active (non-dissolved) state before executing the operation. This allows any authorized principal to stake maturity on a dissolved neuron — a "closed" entity — which is semantically invalid and violates the protocol's lifecycle invariant. The NNS Governance canister's equivalent function correctly rejects this operation.

### Finding Description

In `rs/sns/governance/src/governance.rs`, the `stake_maturity_of_neuron` function performs only two checks before mutating neuron state: authorization and percentage validity. [1](#0-0) 

There is no check for the neuron's dissolve state. A dissolved SNS neuron (one where `when_dissolved_timestamp_seconds` is in the past, or `dissolve_delay_seconds == 0`) can have maturity staked into `staked_maturity_e8s_equivalent` by any caller holding the `StakeMaturity` permission.

By direct contrast, the NNS Governance canister's equivalent function explicitly rejects this operation on dissolved neurons: [2](#0-1) 

The SNS neuron lifecycle defines three states — `NotDissolving`, `Dissolving`, and `Dissolved` — and the protocol's invariant is that dissolved neurons have their `staked_maturity_e8s_equivalent` automatically moved to regular `maturity_e8s_equivalent` by the periodic task `maybe_move_staked_maturity`: [3](#0-2) 

The `StakeMaturity` command is explicitly listed as allowed through the `manage_neuron_internal` dispatch path with no dissolved-state guard: [4](#0-3) 

### Impact Explanation

An authorized principal (controller or hotkey with `StakeMaturity` permission) can call `manage_neuron` with a `StakeMaturity` command on a dissolved SNS neuron. This:

1. **Violates the protocol lifecycle invariant**: Dissolved neurons are "closed" — their staked maturity is supposed to be automatically converted to regular maturity. Allowing new staked maturity to be added to a dissolved neuron contradicts this invariant.
2. **Creates a persistent state inconsistency**: The staked maturity added to a dissolved neuron will be moved back to regular maturity only when the periodic task `maybe_move_staked_maturity` runs. Between the call and the periodic task execution, the neuron's accounting is incorrect.
3. **Potential reward accounting distortion**: If reward calculations snapshot `staked_maturity_e8s_equivalent` before the periodic cleanup runs, a dissolved neuron could receive inflated reward consideration it is not entitled to.
4. **Misleading API response**: The call succeeds and returns a `StakeMaturityResponse` indicating the operation completed, when the protocol intends it to be rejected. [5](#0-4) 

### Likelihood Explanation

**High likelihood of triggering**: Any principal holding the `StakeMaturity` permission on a dissolved SNS neuron can trigger this with a standard `manage_neuron` ingress call. No privileged access, no threshold corruption, and no external dependency is required. The SNS governance canister is publicly reachable. Dissolved neurons are a normal end-state in the SNS lifecycle.

### Recommendation

Add a dissolved-state check at the top of `stake_maturity_of_neuron` in `rs/sns/governance/src/governance.rs`, mirroring the NNS Governance implementation:

```rust
let neuron_state = neuron.state(self.env.now());
if neuron_state == NeuronState::Dissolved {
    return Err(GovernanceError::new_with_message(
        ErrorType::PreconditionFailed,
        "Can't perform operation on neuron: Neuron is dissolved.",
    ));
}
```

This aligns the SNS implementation with the NNS implementation and enforces the protocol invariant that dissolved neurons cannot have maturity staked into them. [6](#0-5) 

### Proof of Concept

1. Create an SNS neuron with maturity and a dissolve delay of 1 second.
2. Wait for the neuron to dissolve (advance time past `when_dissolved_timestamp_seconds`).
3. Verify the neuron is in `Dissolved` state via `neuron.state(now)`.
4. Call `manage_neuron` with `Command::StakeMaturity { percentage_to_stake: Some(100) }` as the authorized controller.
5. Observe: the call **succeeds** and returns `StakeMaturityResponse { maturity_e8s: 0, staked_maturity_e8s: <original_maturity> }`.
6. Observe: the dissolved neuron now has `staked_maturity_e8s_equivalent > 0`, violating the protocol invariant.
7. Compare: the same call on NNS Governance returns `ErrorType::PreconditionFailed` with message `"Can't perform operation on neuron: Neuron is dissolved."`. [7](#0-6)

    pub fn stake_maturity_of_neuron(
        &mut self,
        id: &NeuronId,
        caller: &PrincipalId,
        stake_maturity: &manage_neuron::StakeMaturity,
    ) -> Result<StakeMaturityResponse, GovernanceError> {
        let neuron = self.get_neuron_result(id)?.clone();

        let nid = neuron.id.as_ref().expect("Neurons must have an id");

        if !neuron.is_authorized(caller, NeuronPermissionType::StakeMaturity) {
            return Err(GovernanceError::new(ErrorType::NotAuthorized));
        }

        let percentage_to_stake = stake_maturity.percentage_to_stake.unwrap_or(100);

        if percentage_to_stake > 100 || percentage_to_stake == 0 {
            return Err(GovernanceError::new_with_message(
                ErrorType::PreconditionFailed,
                "The percentage of maturity to stake must be a value between 0 (exclusive) and 100 (inclusive).",
            ));
        }

        let mut maturity_to_stake = (neuron
            .maturity_e8s_equivalent
            .saturating_mul(percentage_to_stake as u64))
            / 100;

        if maturity_to_stake > neuron.maturity_e8s_equivalent {
            maturity_to_stake = neuron.maturity_e8s_equivalent;
        }

        // Adjust the maturity of the neuron
        let neuron = self
            .get_neuron_result_mut(nid)
            .expect("Expected the neuron to exist");

        neuron.maturity_e8s_equivalent = neuron
            .maturity_e8s_equivalent
            .saturating_sub(maturity_to_stake);

        neuron.staked_maturity_e8s_equivalent = Some(
            neuron
                .staked_maturity_e8s_equivalent
                .unwrap_or(0)
                .saturating_add(maturity_to_stake),
        );

        Ok(StakeMaturityResponse {
            maturity_e8s: neuron.maturity_e8s_equivalent,
            staked_maturity_e8s: neuron.staked_maturity_e8s_equivalent.unwrap_or(0),
        })
    }

    pub fn state(&self, now_seconds: u64) -> NeuronState {
        match self.dissolve_state {
            Some(DissolveState::DissolveDelaySeconds(d)) => {
                if d > 0 {
                    NeuronState::NotDissolving
                } else {
                    NeuronState::Dissolved
                }
            }
            Some(DissolveState::WhenDissolvedTimestampSeconds(ts)) => {
                if ts > now_seconds {
                    NeuronState::Dissolving
                } else {
                    NeuronState::Dissolved
                }
            }
            None => NeuronState::Dissolved,
        }
