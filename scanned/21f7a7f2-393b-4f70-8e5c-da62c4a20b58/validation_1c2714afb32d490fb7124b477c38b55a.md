### Title
SNS Governance Mode Locked to Swap Canister — Governance Cannot Self-Recover via Proposal - (File: `rs/sns/governance/src/governance.rs`, `rs/sns/governance/canister/canister.rs`)

### Summary
The `set_mode` endpoint of the SNS Governance canister can only be called by the swap canister. No governance proposal action exists to change the governance mode. If the swap canister fails to finalize and call `set_mode`, the SNS governance is permanently locked in `PreInitializationSwap` mode with no on-chain recovery path through governance proposals — an exact structural analog to the VeloGovernor finding where `setProposalNumerator`/`setTeam` were callable only by `team`, not by `address(this)`.

### Finding Description

In `rs/sns/governance/src/governance.rs`, the `set_mode` function enforces that only the swap canister may call it:

```rust
pub fn set_mode(&mut self, mode: i32, caller: PrincipalId) {
    let mode = governance::Mode::try_from(mode)
        .unwrap_or_else(|_| panic!("Unknown mode: {mode}"));

    if !self.is_swap_canister(caller) {
        panic!("Caller must be the swap canister.");
    }

    if mode != governance::Mode::Normal {
        panic!("Entering {mode:?} mode is not allowed.");
    }

    self.proto.mode = mode as i32;
}
``` [1](#0-0) 

The canister endpoint in `rs/sns/governance/canister/canister.rs` exposes this as a public update method with no additional guard beyond the internal `is_swap_canister` check:

```rust
/// Sets the mode. Only the swap canister is allowed to call this.
#[update]
fn set_mode(request: SetMode) -> SetModeResponse {
    log!(INFO, "set_mode");
    governance_mut().set_mode(request.mode, caller());
    SetModeResponse {}
}
``` [2](#0-1) 

Inspecting all proposal `Action` variants handled in `rs/sns/governance/src/proposal.rs`, there is no `SetMode` action — governance has no proposal path to change its own mode: [3](#0-2) 

The swap canister calls `set_mode` only during `finalize_inner`, specifically via `set_sns_governance_to_normal_mode`: [4](#0-3) 

If that call fails or is never reached, the SNS governance remains in `PreInitializationSwap` mode indefinitely.

### Impact Explanation

When governance is in `PreInitializationSwap` mode, the following proposal actions are explicitly blocked: [5](#0-4) 

This means `ManageNervousSystemParameters`, `UpgradeSnsControlledCanister`, `RegisterDappCanisters`, `DeregisterDappCanisters`, `TransferSnsTreasuryFunds`, and `MintSnsTokens` are all permanently unavailable to the SNS community. The SNS is effectively bricked at the governance layer with no self-healing proposal path. The only external recovery would require NNS governance to upgrade the SNS governance canister's Wasm — a slow, high-friction process that bypasses the SNS community entirely.

### Likelihood Explanation

The swap canister's `finalize` is a public endpoint callable by any principal. If the swap canister encounters a bug during finalization that prevents `set_sns_governance_to_normal_mode` from succeeding, governance is stuck. While finalization can be retried, a persistent bug in the swap canister (e.g., a failed inter-canister call to SNS governance that always errors) would permanently block the mode transition. The swap canister is a trusted system canister, but the absence of a governance-controlled recovery path means any failure in that single call path has unbounded impact duration. Likelihood is **low** but impact is **high** (full governance lockout of an SNS instance).

### Recommendation

Add a `SetMode` native proposal action to SNS governance, callable only when the proposal is executed by the governance canister itself (i.e., `caller == self.env.canister_id()`). This mirrors the judge's recommendation in the VeloGovernor finding: the check should be for `address(this)` (the governance canister's own principal), not a fixed external canister. This provides a governance-controlled recovery path if the swap canister fails to transition the mode.

### Proof of Concept

1. An SNS is deployed and enters `PreInitializationSwap` mode.
2. The swap canister's `finalize` is called but `set_sns_governance_to_normal_mode` fails persistently (e.g., due to a bug in the swap canister or a transient error that becomes permanent).
3. Any attempt to submit a `ManageNervousSystemParameters` or `UpgradeSnsControlledCanister` proposal is rejected with `PreconditionFailed` citing `PreInitializationSwap` mode.
4. No governance proposal action exists to change the mode — confirmed by the absence of `SetMode` in the `validate_and_render_action` match in `rs/sns/governance/src/proposal.rs`.
5. The SNS community is permanently locked out of all critical governance operations with no on-chain recovery path. [6](#0-5)

    fn proposal_action_is_allowed_in_pre_initialization_swap_or_err(
        action: &Action,
        disallowed_target_canister_ids: &HashSet<CanisterId>,
        id_to_nervous_system_function: &BTreeMap<u64, NervousSystemFunction>,
    ) -> Result<(), GovernanceError> {
        // ExecuteGenericNervousSystemFunction is special in that it
        // is only disallowed in some cases.
        if let Action::ExecuteGenericNervousSystemFunction(execute) = action {
            return Self::execute_generic_nervous_system_function_is_allowed_in_pre_initialization_swap_or_err(
                    execute,
                    disallowed_target_canister_ids,
                    id_to_nervous_system_function,
                );
        }

        let nervous_system_function = NervousSystemFunction::from(action.clone());

        let is_action_disallowed = Self::functions_disallowed_in_pre_initialization_swap()
            .into_iter()
            .any(|t| t.id == nervous_system_function.id);

        if is_action_disallowed {
            Err(GovernanceError::new_with_message(
                ErrorType::PreconditionFailed,
                format!(
                    "Proposal type for {:?} is not allowed while governance is in \
                     PreInitializationSwap ({}) mode.",
                    nervous_system_function,
                    Mode::PreInitializationSwap as i32,
                ),
            ))
        } else {
            Ok(())
