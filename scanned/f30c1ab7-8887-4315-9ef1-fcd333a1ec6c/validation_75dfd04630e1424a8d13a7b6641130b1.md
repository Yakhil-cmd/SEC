### Title
SNS Developer Neuron Stake Permanently Locked When Decentralization Swap Aborts - (`rs/sns/governance/src/types.rs`)

### Summary

When an SNS decentralization swap is aborted (analogous to a "denied" position in the reference report), the SNS governance canister remains in `PreInitializationSwap` mode indefinitely. In this mode, `StartDissolving`, `Disburse`, `Split`, `MergeMaturity`, and `DisburseMaturity` are all blocked for every neuron. Developer neurons created at SNS genesis with staked tokens therefore have their stake permanently locked with no protocol-level path to recovery, mirroring the M-05 pattern of a denial action creating a permanent cooldown that prevents asset withdrawal.

### Finding Description

At SNS initialization, developer neurons are created with a `dissolve_delay_seconds` and an optional `vesting_period_seconds`. These neurons hold real staked SNS governance tokens on the ledger. The SNS governance canister starts in `PreInitializationSwap` mode and is only transitioned to `Normal` mode by `set_sns_governance_to_normal_mode`, which is called inside `finalize_inner` exclusively on the **committed** path. [1](#0-0) 

When the swap is **aborted**, `finalize_inner` takes the early-return path after `restore_dapp_controllers_for_finalize` and never calls `set_sns_governance_to_normal_mode`. [2](#0-1) 

The governance mode therefore stays at `PreInitializationSwap`. The `allows_manage_neuron_command_or_err` function enforces that in this mode only `Follow`, `MakeProposal`, `RegisterVote`, `AddNeuronPermissions`, and `RemoveNeuronPermissions` are permitted. Every other command — including `Configure` (which contains `StartDissolving`), `Disburse`, `Split`, `MergeMaturity`, and `DisburseMaturity` — is rejected. [3](#0-2) 

Because `StartDissolving` is blocked, a developer neuron can never transition from `NotDissolving` to `Dissolving`, and therefore can never reach `Dissolved`. Because `Disburse` requires `NeuronState::Dissolved`, the staked tokens can never be retrieved. [4](#0-3) 

The integration test suite explicitly confirms this behavior: after a swap abort, `start_dissolving_neuron` returns a `PreconditionFailed` error containing `"PreInitializationSwap"`. [5](#0-4) 

### Impact Explanation

Developer neurons created at SNS genesis hold real staked SNS governance tokens. If the decentralization swap aborts, those tokens are permanently locked: the owner cannot start dissolving, cannot disburse, cannot split, and cannot disburse maturity. There is no governance proposal type that transitions the SNS governance mode back to `Normal` after an aborted swap, and no canister upgrade hook is automatically triggered to do so. The funds are not lost (they remain on the ledger in the neuron's subaccount) but are inaccessible for an unbounded period — exactly the medium-severity "locked but not lost" impact class of M-05.

### Likelihood Explanation

An SNS swap aborts whenever the minimum participation thresholds (`min_participants`, `min_icp_e8s`, `min_direct_participation_icp_e8s`) are not met before `swap_due_timestamp_seconds`. This is a normal, expected lifecycle outcome reachable by any combination of unprivileged direct participants simply not contributing enough ICP. No privileged access, key compromise, or majority attack is required. Any SNS project whose swap fails to reach minimum participation will trigger this condition for all developer neurons.

### Recommendation

In `finalize_inner`, call `set_sns_governance_to_normal_mode` on the **aborted** path as well, or introduce a dedicated `set_sns_governance_to_pre_initialization_swap_failed_mode` that lifts the neuron-management restrictions while keeping other `PreInitializationSwap` guards (e.g., treasury transfers) in place. Alternatively, add a separate `release_developer_neurons` endpoint callable only by the swap canister after an abort that explicitly allows `StartDissolving` and `Disburse` for developer neurons.

### Proof of Concept

1. An SNS is initialized; developer neurons are created with `dissolve_delay_seconds = ONE_YEAR` and staked tokens.
2. The decentralization swap opens but closes without meeting `min_participants`.
3. `try_abort` is called; `lifecycle` transitions to `Aborted`.
4. `finalize` is called; `finalize_inner` executes `sweep_icp` (refunds direct participants), calls `settle_neurons_fund_participation`, then calls `restore_dapp_controllers_for_finalize` and **returns early** — `set_sns_governance_to_normal_mode` is never called.
5. The developer neuron owner calls `manage_neuron` with `Command::Configure(StartDissolving)`.
6. `allows_manage_neuron_command_or_err` is invoked with `Mode::PreInitializationSwap`; `Configure` is not in the allowed list; the call returns `PreconditionFailed: "Because governance is currently in PreInitializationSwap mode, manage_neuron commands of this type are not allowed"`.
7. The developer neuron owner calls `manage_neuron` with `Command::Disburse`; same rejection.
8. The staked tokens remain locked indefinitely with no protocol path to recovery. [6](#0-5) [7](#0-6) [8](#0-7)

    pub async fn finalize_inner(
        &mut self,
        now_fn: fn(bool) -> u64,
        environment: &mut impl CanisterEnvironment,
    ) -> FinalizeSwapResponse {
        let mut finalize_swap_response = FinalizeSwapResponse::default();

        if let Err(e) = self.can_finalize() {
            finalize_swap_response.set_error_message(e);
            return finalize_swap_response;
        }

        // Transfer the ICP tokens from the Swap canister.
        finalize_swap_response
            .set_sweep_icp_result(self.sweep_icp(now_fn, environment.icp_ledger()).await);
        if finalize_swap_response.has_error_message() {
            return finalize_swap_response;
        }

        // Settle the Neurons' Fund participation in the token swap.
        finalize_swap_response.set_settle_neurons_fund_participation_result(
            self.settle_neurons_fund_participation(environment.nns_governance_mut())
                .await,
        );
        if finalize_swap_response.has_error_message() {
            return finalize_swap_response;
        }

        if self.should_restore_dapp_control() {
            // Restore controllers of dapp canisters to their original
            // owners (i.e. self.init.fallback_controller_principal_ids).
            finalize_swap_response.set_set_dapp_controllers_result(
                self.restore_dapp_controllers_for_finalize(environment.sns_root_mut())
                    .await,
            );

            // In the case of returning control of the dapp(s) to the fallback
            // controllers, finalize() need not do any more work, so always return
            // and end execution.
            return finalize_swap_response;
        }

impl governance::Mode {
    pub fn allows_manage_neuron_command_or_err(
        &self,
        command: &manage_neuron::Command,
        caller_is_swap_canister: bool,
    ) -> Result<(), GovernanceError> {
        use governance::Mode;
        match self {
            Mode::Unspecified => panic!("Governance's mode is not specified."),
            Mode::Normal => Ok(()),
            Mode::PreInitializationSwap => {
                Self::manage_neuron_command_is_allowed_in_pre_initialization_swap_or_err(
                    command,
                    caller_is_swap_canister,
                )
            }
        }
    }

    fn manage_neuron_command_is_allowed_in_pre_initialization_swap_or_err(
        command: &manage_neuron::Command,
        caller_is_swap_canister: bool,
    ) -> Result<(), GovernanceError> {
        use manage_neuron::Command as C;
        let ok = match command {
            C::Follow(_)
            | C::MakeProposal(_)
            | C::RegisterVote(_)
            | C::AddNeuronPermissions(_)
            | C::RemoveNeuronPermissions(_) => true,

            C::ClaimOrRefresh(_) => caller_is_swap_canister,

            _ => false,
        };

        if ok {
            return Ok(());
        }

        Err(GovernanceError::new_with_message(
            ErrorType::PreconditionFailed,
            format!(
                "Because governance is currently in PreInitializationSwap mode, \
                 manage_neuron commands of this type are not allowed \
                 (caller_is_swap_canister={caller_is_swap_canister}). command: {command:#?}",
            ),
        ))
    }
