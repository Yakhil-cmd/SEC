Audit Report

## Title
Staked Maturity Permanently Locked in Inactive Dissolved Neuron — (`rs/nns/governance/src/neuron/types.rs`, `rs/nns/governance/src/neuron_store.rs`)

## Summary

`is_funded()` omits `staked_maturity_e8s_equivalent` from its funded-amount calculation. A dissolved neuron with only staked maturity (zero ICP stake, zero unstaked maturity) that dissolved more than 2 weeks ago is classified as inactive, filtered out of `with_active_neurons_iter_sections`, and therefore never reached by `list_neurons_ready_to_unstake_maturity`. The timer task — the sole mechanism to convert staked maturity to spendable maturity — can never process it, permanently stranding the user's funds.

## Finding Description

**Root cause — `is_funded()` ignores staked maturity:** [1](#0-0) 

Only `stake_e8s()` (`cached_neuron_stake_e8s − neuron_fees_e8s`) and `maturity_e8s_equivalent` are summed. `staked_maturity_e8s_equivalent` is absent.

**`is_inactive()` uses `is_funded()` as its funded gate:** [2](#0-1) 

A neuron with `staked_maturity_e8s_equivalent > 0`, `cached_neuron_stake_e8s == 0`, and `maturity_e8s_equivalent == 0` passes the "not funded" gate and, once dissolved ≥ 2 weeks, becomes inactive.

**Active-neuron iterator filters inactive neurons:** [3](#0-2) 

**`list_neurons_ready_to_unstake_maturity` is built on top of this filtered iterator:** [4](#0-3) 

The affected neuron is never returned, never passed to `unstake_maturity`, and its staked maturity is permanently stranded.

**The existing proptest already encodes the correct (but unimplemented) invariant:** [5](#0-4) 

`net_funding_e8s` includes `staked_maturity_e8s_equivalent`, and the test asserts `neuron.is_inactive(now) == !is_funded`. This proptest will fail for any case where `staked_maturity_e8s_equivalent > 0` and all other funding fields are zero — directly demonstrating the implementation diverges from its own specification.

**The timer task comment confirms the invariant is expected to hold:** [6](#0-5) 

## Impact Explanation

A neuron controller who disburses their ICP stake after dissolution while retaining `staked_maturity_e8s_equivalent > 0` will, after 2 weeks, have that staked maturity permanently inaccessible. The timer task is the **only** mechanism to convert staked maturity to regular maturity; there is no user-callable governance method to trigger it manually. The controller permanently loses the ability to spawn that maturity into ICP. This constitutes a concrete, permanent loss of NNS governance assets (staked maturity denominated in ICP e8s) reachable through entirely normal governance flows. This matches the **High** impact class: *Significant NNS security impact with concrete user or protocol harm* / *Unauthorized access to neurons, governance assets … or canister-controlled funds*.

## Likelihood Explanation

No privileged access is required. Any neuron controller can reach this state through the standard sequence: dissolve neuron → disburse ICP stake → wait 2 weeks. The controller harms only themselves, but the path is fully unprivileged and requires no special conditions beyond holding a dissolved neuron with staked maturity. The proptest in the test suite already generates the triggering inputs automatically.

## Recommendation

Include `staked_maturity_e8s_equivalent` in `is_funded()`:

```rust
pub fn is_funded(&self) -> bool {
    let amount_e8s = self
        .stake_e8s()
        .saturating_add(self.staked_maturity_e8s_equivalent.unwrap_or(0))
        .saturating_add(self.maturity_e8s_equivalent);
    amount_e8s > 0
}
```

This aligns the implementation with the invariant already expressed in the proptest at `rs/nns/governance/src/governance/tests/mod.rs` lines 1121–1128.

## Proof of Concept

1. Create a neuron with `cached_neuron_stake_e8s = 0`, `neuron_fees_e8s = 0`, `maturity_e8s_equivalent = 0`, `staked_maturity_e8s_equivalent = Some(1_000_000)`, dissolved at `now − 3 * 7 * ONE_DAY_SECONDS`.
2. Assert `neuron.is_inactive(now) == true` — confirmed because `is_funded()` returns `false`.
3. Construct a `NeuronStore` containing only this neuron.
4. Call `neuron_store.unstake_maturity_of_dissolved_neurons(now, usize::MAX)` repeatedly.
5. Assert `neuron.staked_maturity_e8s_equivalent == Some(1_000_000)` — it never changes.
6. The existing proptest `test_neuron_is_inactive_based_on_funding` at lines 1113–1160 already generates this exact case and will fail against the current `is_funded()` implementation, serving as a ready-made reproducible proof.

        // Require condition 1.
        if self.is_seed_neuron() || self.is_ect_neuron() {
            return false;
        }

        // Require condition 2.
        if self.is_funded() {
            return false;
        }

        // Require condition 3.

        // 3.1: Interpret dissolve_state field.
        let dissolved_at_timestamp_seconds = match self.dissolved_at_timestamp_seconds() {
            // None -> not dissolving -> will be dissolved in the future -> not dissolved now ->
            // certainly was not dissolved sufficiently "long" ago!
            None => {
                return false;
            }
            Some(ok) => ok,
        };

        // 3.2: Now, we know when self is "dissolved" (could be in the past, present, or future).
        // Thus, we can evaluate whether that happened sufficiently long ago.

            now.saturating_sub(2 * 7 * ONE_DAY_SECONDS);

            return false;
        }

        // Finally, require condition 4: Member of the Neuron's Fund.
        if self.is_a_neurons_fund_member() {
            return false;
        }

        // All requirements have been met.
        true
    }
