Looking at the exact code path in `calculate_merge_neurons_effect` and the voting power calculation to determine if the invariant can actually break.

### Title
Unconditional `eight_year_gang_bonus_base_e8s` Transfer in `calculate_merge_neurons_effect` When No Stake Moves — (`rs/nns/governance/src/governance/merge_neurons.rs`)

---

### Summary

In `calculate_merge_neurons_effect`, `transfer_eight_year_gang_bonus_base_e8s` is assigned unconditionally from `source.eight_year_gang_bonus_base_e8s` at line 269, with no guard on whether `stake_transfer_to_target_e8s` is `Some` or `None`. When the source neuron's `minted_stake_e8s` is at or below `transaction_fees_e8s`, no ICP moves on-ledger, yet the full bonus base is still subtracted from the source and added to the target. Because the voting power formula caps the bonus base to `stake_e8s`, a source with near-zero stake had a near-zero effective bonus; after the merge the target — which may have a large stake — gains the full uncapped bonus. This is a net increase in total system voting power with no corresponding ICP transfer.

---

### Finding Description

**Root cause — line 269:**

```rust
// rs/nns/governance/src/governance/merge_neurons.rs
let amount_to_target_e8s = source.minted_stake_e8s.saturating_sub(transaction_fees_e8s);
let stake_transfer_to_target_e8s = if amount_to_target_e8s > 0 {
    Some(amount_to_target_e8s)
} else {
    None                          // ← no stake moves
};
let transfer_eight_year_gang_bonus_base_e8s = source.eight_year_gang_bonus_base_e8s; // ← unconditional
``` [1](#0-0) 

The field is then applied unconditionally in both `source_effect()` and `target_effect()`:

```rust
subtract_eight_year_gang_bonus_base_e8s: self.transfer_eight_year_gang_bonus_base_e8s,
// ...
add_eight_year_gang_bonus_base_e8s: self.transfer_eight_year_gang_bonus_base_e8s,
``` [2](#0-1) 

The `apply` methods on `MergeNeuronsSourceEffect` and `MergeNeuronsTargetEffect` then mutate the neurons: [3](#0-2) 

**Voting power formula — the cap that makes this exploitable:**

```rust
// rs/nns/governance/src/neuron/types.rs
let eight_year_gang_bonus_base_e8s = self.eight_year_gang_bonus_base_e8s.min(stake_e8s);
potential_voting_power +=
    Decimal::from(eight_year_gang_bonus_base_e8s) / Decimal::from(10) * boost;
``` [4](#0-3) 

Because the bonus is capped to `stake_e8s`, a source with `minted_stake_e8s = 9_000 e8s` and `eight_year_gang_bonus_base_e8s = 100 ICP` contributes only `900 e8s / 10 * boost` of bonus voting power. After the merge, the target (with e.g. 100 ICP stake) gains `min(100 ICP, 100 ICP) / 10 * boost = 10 ICP * boost` of bonus voting power — a ~11,111× amplification of the effective bonus with zero ICP transferred.

**The existing tests do not cover this case.** Both `test_calculate_effect_no_stake_transfer` and `test_calculate_effect_no_stake_transfer_or_burn_fees` build source neurons without setting `eight_year_gang_bonus_base_e8s`, so it defaults to 0 and the tests pass trivially: [5](#0-4) [6](#0-5) 

**How a source neuron reaches the triggering state:**

`eight_year_gang_bonus_base_e8s` is set at migration time to `(cached_stake − fees) + staked_maturity` for neurons that held the 8-year maximum dissolve delay: [7](#0-6) 

It is zeroed only when the neuron starts dissolving: [8](#0-7) 

A `NotDissolving` neuron that accumulated `neuron_fees_e8s` close to its `cached_neuron_stake_e8s` through rejected proposals will have `minted_stake_e8s ≤ transaction_fees_e8s` while retaining its full `eight_year_gang_bonus_base_e8s`. This is a naturally reachable state, not one that requires special privilege.

---

### Impact Explanation

An unprivileged principal who controls both neurons can call `manage_neuron` → `Merge` via the standard governance ingress path. When the precondition holds, the target neuron's `eight_year_gang_bonus_base_e8s` is inflated by the full source value. Because the voting power formula applies a 10% bonus on top of the capped base, the target gains up to `target.stake_e8s / 10 * boost` additional voting power that was not present in the system before the merge. This inflates the target's share of NNS governance voting power and voting rewards without any ICP being transferred.

---

### Likelihood Explanation

The triggering state (8-year gang neuron with near-zero `minted_stake_e8s`) is uncommon but naturally reachable for any legacy 8-year gang member whose neuron accumulated substantial rejection fees. No privileged access, key compromise, or majority attack is required — only ownership of both neurons and a standard `manage_neuron` call.

---

### Recommendation

Guard the assignment at line 269 on whether stake actually moves:

```rust
let transfer_eight_year_gang_bonus_base_e8s = if stake_transfer_to_target_e8s.is_some() {
    source.eight_year_gang_bonus_base_e8s
} else {
    0
};
```

Add a test that sets `eight_year_gang_bonus_base_e8s > 0` on the source neuron in the `test_calculate_effect_no_stake_transfer` and `test_calculate_effect_no_stake_transfer_or_burn_fees` cases and asserts `transfer_eight_year_gang_bonus_base_e8s == 0`.

fn test_eight_year_gang_bonus_base_e8s_is_lost_after_dissolving() {
    let now = 123_456_789;
    let dissolve_delay_seconds = 8 * ONE_YEAR_SECONDS;
    let mut neuron = NeuronBuilder::new(
        NeuronId { id: 1 },
        Subaccount::try_from(vec![0_u8; 32].as_slice()).unwrap(),
        PrincipalId::new_user_test_id(1),
        DissolveStateAndAge::NotDissolving {
            dissolve_delay_seconds,
            aging_since_timestamp_seconds: now - 1000,
        },
        now - 2000,
    )
    .with_cached_neuron_stake_e8s(100 * E8)
    .with_eight_year_gang_bonus_base_e8s(100 * E8)
    .build();
    let controller = neuron.controller();

    // Verify the neuron is not dissolving and has a non-zero eight year gang bonus.
    assert_eq!(neuron.state(now), NeuronState::NotDissolving);
    assert_eq!(neuron.eight_year_gang_bonus_base_e8s, 100 * E8);

    // Start dissolving.
    neuron
        .configure(
            &controller,
            now,
            &Configure {
                operation: Some(Operation::StartDissolving(StartDissolving {})),
            },
        )
        .unwrap();

    // Verify the neuron is dissolving and the eight year gang bonus has been set to 0.
    assert_eq!(neuron.state(now), NeuronState::Dissolving);
    assert_eq!(neuron.eight_year_gang_bonus_base_e8s, 0);
