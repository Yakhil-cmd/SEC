### Title
Silo Fixed-Gas Fee and Whitelist Enforcement Bypassed via `call` Entrypoint — (`File: engine/src/contract_methods/evm_transactions.rs`)

### Summary
In Silo mode, Aurora Engine enforces a `fixed_gas` fee and an account/address whitelist on every EVM transaction submitted through `submit` and `submit_with_args`. The `call` NEAR entrypoint — which is publicly accessible to any NEAR account — performs neither check, allowing an unprivileged user to execute arbitrary EVM calls without paying the Silo fee and without being on the whitelist.

### Finding Description

The `submit` path in `engine/src/engine.rs` enforces two Silo-mode controls before executing any EVM transaction:

1. **Whitelist check** via `assert_access(&io, env, &transaction)` — verifies the NEAR caller account (`WhitelistKind::Account`) and the EVM sender address (`WhitelistKind::Address`) are both whitelisted.
2. **Fixed-gas fee** via `engine.charge_gas(&sender, &transaction, max_gas_price, fixed_gas)` — deducts `fixed_gas * gas_price` from the EVM sender's balance and credits the relayer. [1](#0-0) 

The `call` entrypoint in `engine/src/contract_methods/evm_transactions.rs` (exposed as the public NEAR method `call()` in `engine/src/lib.rs`) does none of this. It reads state, constructs an `Engine`, and immediately calls `engine.call_with_args(args, handler)` with no whitelist check and no fixed-gas deduction: [2](#0-1)

### Impact Explanation

**Whitelist bypass (High):** Any NEAR account — even one not on the `WhitelistKind::Account` or `WhitelistKind::Address` lists — can call EVM contracts in a Silo by using the `call` entrypoint. The Silo operator's intent to restrict EVM access to approved participants is completely defeated.

**Fee bypass (High — theft of unclaimed yield):** The `fixed_gas` fee is the mechanism by which Silo operators/relayers collect revenue per EVM transaction. By routing through `call` instead of `submit`, a user pays zero fixed-gas fee. The relayer never receives the `priority_fee_per_gas * fixed_gas` reward that `refund_unused_gas` would otherwise credit to it. [6](#0-5) 

### Likelihood Explanation

**High.** The `call` NEAR method is a standard, documented public entrypoint on the Aurora Engine contract. Any NEAR account can call it with a crafted `CallArgs` payload. No special privileges, leaked keys, or governance capture are required. The attacker only needs a NEAR account and knowledge of the target EVM contract address and calldata.

### Recommendation

Apply the same Silo-mode guards inside `call` (and `deploy_code`) that are applied inside `submit`:

1. Read `silo::get_fixed_gas` and, if set, deduct the fee from the EVM address derived from `predecessor_account_id` before executing.
2. Call `assert_access` (or an equivalent check) to verify the NEAR caller and derived EVM address are whitelisted before proceeding.

Alternatively, document explicitly that `call` is intentionally exempt from Silo controls and restrict it to the contract owner/operator via a predecessor-account check.

### Proof of Concept

1. Operator deploys Aurora Engine in Silo mode: sets `fixed_gas = 1_000_000` and enables `WhitelistKind::Account` and `WhitelistKind::Address` with no entries.
2. Attacker (NEAR account `attacker.near`, not whitelisted) attempts `submit` with a signed EVM transaction → receives `EngineErrorKind::NotAllowed`. [7](#0-6) 

3. Attacker instead calls the `call` NEAR method directly with `CallArgs::V1 { contract: <target_evm_address>, input: <calldata> }`.
4. `call` constructs an `Engine` with sender = `predecessor_address("attacker.near")`, skips `assert_access` and `charge_gas`, and executes the EVM call successfully.
5. The attacker has interacted with the EVM without being whitelisted and without paying the fixed-gas fee. The relayer's balance is not incremented. [2](#0-1)

pub fn refund_unused_gas<I: IO>(
    io: &mut I,
    sender: &Address,
    gas_used: u64,
    gas_result: &GasPaymentResult,
    relayer: &Address,
    fixed_gas: Option<EthGas>,
) -> Result<(), GasPaymentError> {
    if gas_result.effective_gas_price.is_zero() {
        return Ok(());
    }

    let (refund, relayer_reward) = {
        let gas_to_wei = |price: U256| {
            fixed_gas
                .map_or_else(|| gas_used.into(), EthGas::as_u256)
                .checked_mul(price)
                .map(Wei::new)
                .ok_or(GasPaymentError::EthAmountOverflow)
        };

        let spent_amount = gas_to_wei(gas_result.effective_gas_price)?;
        let reward_amount = gas_to_wei(gas_result.priority_fee_per_gas)?;

        let refund = gas_result
            .prepaid_amount
            .checked_sub(spent_amount)
            .ok_or(GasPaymentError::EthAmountOverflow)?;

        (refund, reward_amount)
    };

    if !refund.is_zero() {
        add_balance(io, sender, refund)?;
    }

    if !relayer_reward.is_zero() {
        add_balance(io, relayer, relayer_reward)?;
    }

pub fn call<I: IO + Copy, E: Env, H: PromiseHandler>(
    io: I,
    env: &E,
    handler: &mut H,
) -> Result<SubmitResult, ContractError> {
    with_logs_hashchain(io, env, function_name!(), |mut io| {
        let state = state::get_state(&io)?;
        require_running(&state)?;
        let bytes = io.read_input().to_vec();
        let args = CallArgs::deserialize(&bytes).ok_or(errors::ERR_BORSH_DESERIALIZE)?;
        let current_account_id = env.current_account_id();
        let predecessor_account_id = env.predecessor_account_id();

        let mut engine: Engine<_, E, AuroraModExp> = Engine::new_with_state(
            state,
            predecessor_address(&predecessor_account_id),
            current_account_id,
            io,
            env,
        );
        let result = engine.call_with_args(args, handler)?;
        let result_bytes = borsh::to_vec(&result).map_err(|_| errors::ERR_SERIALIZE)?;
        io.return_output(&result_bytes);
        Ok(result)
    })
}

/// Return gas amount per transaction.
pub fn get_fixed_gas<I: IO>(io: &I) -> Option<EthGas> {
    let key = fixed_gas_key();
    io.read_storage(&key)
        .and_then(|bytes| bytes.to_value().ok())
}

/// Set an amount of gas per transaction.
pub fn set_fixed_gas<I: IO>(io: &mut I, gas: Option<EthGas>) {
    let key = fixed_gas_key();

    if let Some(gas) = gas {
        io.write_borsh(&key, &gas);
    } else {
        io.remove_storage(&key);
    }
}
