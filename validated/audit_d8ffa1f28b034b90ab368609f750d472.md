### Title
`merge_neurons` Does Not Reconcile `voting_power_refreshed_timestamp_seconds` Across Neurons With Different Refresh Ages — (File: `rs/nns/governance/src/governance/merge_neurons.rs`)

---

### Summary

`MergeNeuronsTargetEffect::apply()` never touches `voting_power_refreshed_timestamp_seconds` on the target neuron. When a stale source neuron (whose deciding voting power has been linearly reduced toward zero because it has not voted or refreshed in >6 months) is merged into a freshly-refreshed target neuron, the combined neuron inherits the target's recent timestamp. The source neuron's entire stake is thereby silently "laundered" to full deciding voting power without the owner ever having to participate in governance.

---

### Finding Description

`deciding_voting_power` is computed in `potential_and_deciding_voting_power()` by multiplying `potential_voting_power` by an `adjustment_factor` derived solely from `self.voting_power_refreshed_timestamp_seconds`: [1](#0-0) 

The adjustment factor is 1.0 for neurons refreshed within 6 months, then decreases linearly to 0.0 over the following month (`DEFAULT_START_REDUCING_VOTING_POWER_AFTER_SECONDS` = 6 months, `DEFAULT_CLEAR_FOLLOWING_AFTER_SECONDS` = 1 month). [2](#0-1) 

When two neurons are merged, `MergeNeuronsTargetEffect::apply()` updates `dissolve_state_and_age`, maturity, staked maturity, and the eight-year-gang bonus base — but **never** `voting_power_refreshed_timestamp_seconds`: [3](#0-2) 

`calculate_merge_neurons_effect()` likewise constructs `MergeNeuronsEffect` with no field for the source neuron's refresh timestamp: [4](#0-3) 

The validation in `validate_request_and_neurons()` checks dissolve state, spawning status, Neurons' Fund membership, managers, KYC, and neuron type — but imposes **no constraint** on the relative staleness of the two neurons' refresh timestamps: [5](#0-4) 

After `merge_neurons()` applies the effect, the target neuron's `voting_power_refreshed_timestamp_seconds` is whatever it was before the merge — the source neuron's staleness is discarded: [6](#0-5) 

---

### Impact Explanation

**Governance authorization bug.** An unprivileged neuron controller can bypass the deciding-voting-power reduction mechanism introduced in Mission 70:

1. Attacker holds a large **stale** neuron S (inactive for ≥7 months → `adjustment_factor = 0`, deciding voting power = 0).
2. Attacker maintains a small **fresh** neuron F (voted recently → `adjustment_factor = 1`).
3. Attacker calls `manage_neuron` → `Merge { source_neuron_id: S }` targeting F.
4. After the merge, F holds the combined stake of S+F but retains F's recent `voting_power_refreshed_timestamp_seconds`.
5. F's `deciding_voting_power` is now `1.0 × potential_voting_power(S+F)` — the full combined stake — instead of the correct `0 × potential_voting_power(S) + 1.0 × potential_voting_power(F)`.

The attacker recovers up to 100 % of the deciding voting power that the staleness penalty was designed to remove, without casting a single vote or calling `RefreshVotingPower` on S. Because deciding voting power governs proposal tallying, a sufficiently large stale neuron merged into a minimal fresh neuron can swing governance outcomes. [7](#0-6) 

---

### Likelihood Explanation

**Medium.** The attacker needs only to:
- Control two neurons under the same principal (standard setup for large ICP holders).
- Allow one neuron to go stale for ≥6 months (trivial — simply stop voting/following).
- Keep a second neuron fresh by voting or calling `RefreshVotingPower` periodically.
- Issue a single `ManageNeuron::Merge` ingress message — a standard, permissionless governance operation.

No privileged access, no threshold corruption, and no social engineering is required. The operation is available to any neuron controller on mainnet.

---

### Recommendation

In `MergeNeuronsTargetEffect` (and its construction in `calculate_merge_neurons_effect`), carry the source neuron's `voting_power_refreshed_timestamp_seconds` and, when applying the effect to the target, set the target's `voting_power_refreshed_timestamp_seconds` to the **minimum** (i.e., the older) of the two timestamps. This ensures the combined neuron's deciding voting power correctly reflects the staleness of the less-active constituent.

Alternatively, prohibit merging neurons whose `voting_power_refreshed_timestamp_seconds` differ by more than `start_reducing_voting_power_after_seconds`, analogous to how the original VotingEscrow report recommends prohibiting merges with different lock-end times.

---

### Proof of Concept

```
// Setup (all on-chain, unprivileged)
let stale_neuron_id  = /* large neuron, last refreshed 7+ months ago, deciding_vp = 0 */;
let fresh_neuron_id  = /* small neuron, refreshed today, deciding_vp = full */;

// Single ingress call
governance.manage_neuron(ManageNeuron {
    id: Some(fresh_neuron_id),
    command: Some(Command::Merge(Merge {
        source_neuron_id: Some(stale_neuron_id),
    })),
});

// After the call:
// fresh_neuron.cached_neuron_stake_e8s  == stale_stake + fresh_stake
// fresh_neuron.voting_power_refreshed_timestamp_seconds == (unchanged, recent)
// fresh_neuron.deciding_voting_power    == 1.0 * potential_vp(stale_stake + fresh_stake)
// Expected (correct) deciding_vp       == 0.0 * potential_vp(stale_stake)
//                                       + 1.0 * potential_vp(fresh_stake)
```

fn validate_request_and_neurons(
    target_neuron_id: &NeuronId,
    merge_neuron: &Merge,
    caller: &PrincipalId,
    neuron_store: &NeuronStore,
    now_seconds: u64,
) -> Result<(ValidSourceNeuron, ValidTargetNeuron), MergeNeuronsError> {
    let source_neuron_id = merge_neuron
        .source_neuron_id
        .ok_or(MergeNeuronsError::NoSourceNeuronId)?;

    if source_neuron_id == *target_neuron_id {
        return Err(MergeNeuronsError::SourceAndTargetSame);
    }

    let (
        source_neuron_to_merge,
        source_is_caller_authorized,
        source_is_not_spawning,
        source_is_not_in_neurons_fund,
        source_neuron_managers,
        source_kyc_verified,
        source_not_for_profit,
        source_neuron_type,
    ) = neuron_store
        .with_neuron(&source_neuron_id, |source_neuron| {
            let source_neuron_to_merge = ValidSourceNeuron::try_new(source_neuron, now_seconds);
            let source_is_caller_authorized =
                source_neuron.is_authorized_to_simulate_manage_neuron(caller);
            let source_is_not_spawning = source_neuron.state(now_seconds) != NeuronState::Spawning;
            let source_is_not_in_neurons_fund = !source_neuron.is_a_neurons_fund_member();
            let source_neuron_managers = source_neuron.neuron_managers();
            let source_kyc_verified = source_neuron.kyc_verified;
            let source_not_for_profit = source_neuron.not_for_profit;
            let source_neuron_type = source_neuron.neuron_type;

            (
                source_neuron_to_merge,
                source_is_caller_authorized,
                source_is_not_spawning,
                source_is_not_in_neurons_fund,
                source_neuron_managers,
                source_kyc_verified,
                source_not_for_profit,
                source_neuron_type,
            )
        })
        .map_err(|_| MergeNeuronsError::SourceNeuronNotFound)?;
    if !source_is_caller_authorized {
        return Err(MergeNeuronsError::SourceNeuronNotHotKeyOrController);
    }
    if !source_is_not_spawning {
        return Err(MergeNeuronsError::SourceNeuronSpawning);
    }
    if !source_is_not_in_neurons_fund {
        return Err(MergeNeuronsError::SourceNeuronInNeuronsFund);
    }
    let source_neuron_to_merge = source_neuron_to_merge?;

    let (
        target_neuron_to_merge,
        target_is_caller_authorized,
        target_is_not_spawning,
        target_is_not_in_neurons_fund,
        target_neuron_managers,
        target_kyc_verified,
        target_not_for_profit,
        target_neuron_type,
    ) = neuron_store
        .with_neuron(target_neuron_id, |target_neuron| {
            let target_neuron_to_merge = ValidTargetNeuron::try_new(target_neuron, now_seconds);
            let target_is_caller_authorized =
                target_neuron.is_authorized_to_simulate_manage_neuron(caller);
            let target_is_not_spawning = target_neuron.state(now_seconds) != NeuronState::Spawning;
            let target_is_not_in_neurons_fund = !target_neuron.is_a_neurons_fund_member();
            let target_neuron_managers = target_neuron.neuron_managers();
            let target_kyc_verified = target_neuron.kyc_verified;
            let target_not_for_profit = target_neuron.not_for_profit;
            let target_neuron_type = target_neuron.neuron_type;

            (
                target_neuron_to_merge,
                target_is_caller_authorized,
                target_is_not_spawning,
                target_is_not_in_neurons_fund,
                target_neuron_managers,
                target_kyc_verified,
                target_not_for_profit,
                target_neuron_type,
            )
        })
        .map_err(|_| MergeNeuronsError::TargetNeuronNotFound)?;
    if !target_is_caller_authorized {
        return Err(MergeNeuronsError::TargetNeuronNotHotKeyOrController);
    }
    if !target_is_not_spawning {
        return Err(MergeNeuronsError::TargetNeuronSpawning);
    }
    if !target_is_not_in_neurons_fund {
        return Err(MergeNeuronsError::TargetNeuronInNeuronsFund);
    }
    let target_neuron_to_merge = target_neuron_to_merge?;

    if source_neuron_managers != target_neuron_managers {
        return Err(MergeNeuronsError::NeuronManagersNotSame);
    }
    if source_kyc_verified != target_kyc_verified {
        return Err(MergeNeuronsError::KycVerifiedNotSame);
    }
    if source_not_for_profit != target_not_for_profit {
        return Err(MergeNeuronsError::NotForProfitNotSame);
    }
    if source_neuron_type != target_neuron_type {
        return Err(MergeNeuronsError::NeuronTypeNotSame);
    }

    Ok((source_neuron_to_merge, target_neuron_to_merge))
}
