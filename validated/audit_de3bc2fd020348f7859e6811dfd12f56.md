Audit Report

## Title
Unconditional `eight_year_gang_bonus_base_e8s` Transfer When No Stake Moves Inflates NNS Voting Power — (`rs/nns/governance/src/governance/merge_neurons.rs`)

## Summary
In `calculate_merge_neurons_effect`, `transfer_eight_year_gang_bonus_base_e8s` is assigned unconditionally from `source.eight_year_gang_bonus_base_e8s` at line 269 regardless of whether `stake_transfer_to_target_e8s` is `Some` or `None`. When a source neuron's `minted_stake_e8s` is at or below `transaction_fees_e8s`, no ICP moves on-ledger, yet the full bonus base is still subtracted from the source and added to the target. Because the voting power formula caps the effective bonus to `stake_e8s`, a source with near-zero stake had near-zero effective bonus voting power; after the merge the target — which may have large stake — gains the full uncapped bonus, creating a net increase in NNS voting power with no corresponding ICP transfer.

## Finding Description

**Root cause — line 269** (`rs/nns/governance/src/governance/merge_neurons.rs`):

```rust
let amount_to_target_e8s = source.minted_stake_e8s.saturating_sub(transaction_fees_e8s);
let stake_transfer_to_target_e8s = if amount_to_target_e8s > 0 {
    Some(amount_to_target_e8s)
} else {
    None   // ← no ICP moves
};
let transfer_eight_year_gang_bonus_base_e8s = source.eight_year_gang_bonus_base_e8s; // ← unconditional
```

The field is then applied unconditionally in both `source_effect()` and `target_effect()`:

- `source_effect()` sets `subtract_eight_year_gang_bonus_base_e8s: self.transfer_eight_year_gang_bonus_base_e8s`
- `target_effect()` sets `add_eight_year_gang_bonus_base_e8s: self.transfer_eight_year_gang_bonus_base_e8s`

Both `MergeNeuronsSourceEffect::apply` and `MergeNeuronsTargetEffect::apply` then mutate the neurons unconditionally via `saturating_sub` / `saturating_add`.

**Voting power formula** (`rs/nns/governance/src/neuron/types.rs`, lines 383–387):

```rust
if is_mission_70_voting_rewards_enabled() {
    let eight_year_gang_bonus_base_e8s = self.eight_year_gang_bonus_base_e8s.min(stake_e8s);
    potential_voting_power +=
        Decimal::from(eight_year_gang_bonus_base_e8s) / Decimal::from(10) * boost;
}
```

`ENABLE_MISSION_70_VOTING_REWARDS` is initialized to `true` in production (`rs/nns/governance/src/lib.rs`, line 223: `= const { Cell::new(true) };`), so this branch is always active.

**The cap creates the amplification**: A source neuron with `minted_stake_e8s = 9_000 e8s` and `eight_year_gang_bonus_base_e8s = 100 ICP` contributes `min(100 ICP, 9_000 e8s) / 10 * boost ≈ 0` bonus voting power. After the merge, the target (with e.g. 100 ICP stake) gains `min(100 ICP, 100 ICP) / 10 * boost = 10 ICP * boost` of bonus voting power — a large amplification with zero ICP transferred.

**Existing tests do not cover this case**: Both `test_calculate_effect_no_stake_transfer` (line 1147) and `test_calculate_effect_no_stake_transfer_or_burn_fees` (line 1266) build source neurons without setting `eight_year_gang_bonus_base_e8s`, so it defaults to 0 and the tests pass trivially. The assertions `transfer_eight_year_gang_bonus_base_e8s: 0` hold only because the source has no bonus base set.

**How the triggering state is reached**: `eight_year_gang_bonus_base_e8s` is set at migration time for neurons that held the 8-year maximum dissolve delay. It is zeroed only when the neuron starts dissolving. A `NotDissolving` neuron that accumulated `neuron_fees_e8s` close to its `cached_neuron_stake_e8s` through rejected proposals will have `minted_stake_e8s ≤ transaction_fees_e8s` while retaining its full `eight_year_gang_bonus_base_e8s`. This is a naturally reachable state.

## Impact Explanation

An unprivileged principal who controls both neurons can call `manage_neuron` → `Merge` via the standard governance ingress path. The target neuron's `eight_year_gang_bonus_base_e8s` is inflated by the full source value with no ICP transferred. Because `is_mission_70_voting_rewards_enabled()` returns `true` in production, the voting power formula applies a 10% bonus on the capped base, giving the target up to `target.stake_e8s / 10 * boost` additional voting power that was not present in the system before the merge. This inflates the attacker's share of NNS governance voting power and voting rewards without any ICP being transferred, constituting a concrete NNS governance integrity impact. This matches: **High ($2,000–$10,000) — Significant NNS security impact with concrete user or protocol harm**.

## Likelihood Explanation

The triggering state (8-year gang neuron with near-zero `minted_stake_e8s`) is uncommon but naturally reachable for any legacy 8-year gang member whose neuron accumulated substantial rejection fees. No privileged access, key compromise, or majority attack is required — only ownership of both neurons and a standard `manage_neuron` call. The attack is repeatable as long as the source neuron retains a non-zero `eight_year_gang_bonus_base_e8s`.

## Recommendation

Guard the assignment at line 269 on whether stake actually moves:

```rust
let transfer_eight_year_gang_bonus_base_e8s = if stake_transfer_to_target_e8s.is_some() {
    source.eight_year_gang_bonus_base_e8s
} else {
    0
};
```

Add tests that set `eight_year_gang_bonus_base_e8s > 0` on the source neuron in both `test_calculate_effect_no_stake_transfer` and `test_calculate_effect_no_stake_transfer_or_burn_fees` and assert `transfer_eight_year_gang_bonus_base_e8s == 0`.

## Proof of Concept

Extend `test_calculate_effect_no_stake_transfer` with a non-zero bonus base:

```rust
let source_neuron = create_model_neuron_builder(1)
    .with_cached_neuron_stake_e8s(10 * E8 + 9_000) // minted_stake = 9_000 < TRANSACTION_FEES_E8S
    .with_neuron_fees_e8s(10 * E8)
    .with_eight_year_gang_bonus_base_e8s(100 * E8)  // ← added
    .with_dissolve_state_and_age(DissolveStateAndAge::NotDissolving { ... })
    .build();
```

Current (buggy) result: `transfer_eight_year_gang_bonus_base_e8s: 100 * E8` — bonus moves with no stake transfer.

Expected (correct) result: `transfer_eight_year_gang_bonus_base_e8s: 0`.

After applying the effect, the target neuron holds `eight_year_gang_bonus_base_e8s = 100 * E8` while receiving zero additional ICP. With `is_mission_70_voting_rewards_enabled()` returning `true`, the voting power formula adds `min(100 ICP, target.stake_e8s) / 10 * boost` to the target's voting power — voting power inflation with no on-chain ICP transfer.
