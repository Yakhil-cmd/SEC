### Title
Any SNS Swap Canister Can Settle Another SNS's Neurons' Fund Participation With Fabricated Data, Due to Insufficient Caller-to-Proposal Binding - (`File: rs/nns/governance/src/governance.rs`)

---

### Summary

`settle_neurons_fund_participation` in NNS Governance only verifies that the caller is **any** deployed SNS swap canister, but never checks that the caller is specifically the swap canister associated with the `nns_proposal_id` supplied in the request. Any SNS swap canister can therefore call this function for a different SNS's proposal, providing fabricated participation figures, causing the NNS to mint incorrect amounts of ICP to the wrong SNS treasury or to incorrectly refund Neurons' Fund maturity.

---

### Finding Description

`settle_neurons_fund_participation` in `rs/nns/governance/src/governance.rs` is documented as callable only by the SNS Swap canister that was created by the `CreateServiceNervousSystem` proposal identified by `nns_proposal_id`:

> "This request may be submitted only by the Swap canister of an SNS instance created by a CreateServiceNervousSystem proposal."

The authorization check at line 7028 calls `is_canister_id_valid_swap_canister_id`, which queries SNS-W's `list_deployed_snses` and checks whether the caller appears as **any** swap canister in the list:

```rust
// Check authorization. Note that a Swap could settle each other's participation.
let target_canister_id: CanisterId = caller.try_into()...?;
if let Err(err_msg) =
    is_canister_id_valid_swap_canister_id(target_canister_id, &*self.env).await
{
    return Err(...);
}
```

`is_canister_id_valid_swap_canister_id` only checks membership in the global set of swap canisters:

```rust
let is_swap = list_deployed_snses_response
    .instances
    .iter()
    .any(|sns| sns.swap_canister_id == Some(target_canister_id.into()));
```

The function never cross-checks that `caller == proposal_data.swap_canister_id` for the specific proposal identified by `nns_proposal_id`. The comment in the code itself acknowledges this gap: *"Note that a Swap could settle each other's participation."*

After passing this check, the function proceeds to:
1. Set the `sns_token_swap_lifecycle` field of the target proposal to `Committed` or `Aborted`
2. Compute Neurons' Fund ICP distribution based on the caller-supplied `total_direct_participation_icp_e8s`
3. Mint ICP to the `sns_governance_canister_id` supplied in the `Committed` variant
4. Refund maturity to Neurons' Fund neurons

All of these state mutations are driven by attacker-controlled fields in the request.

---

### Impact Explanation

A malicious or compromised SNS swap canister (Swap A) can call `settle_neurons_fund_participation` with the `nns_proposal_id` of a different SNS (Swap B) and a fabricated `Committed` result containing:

- An inflated `total_direct_participation_icp_e8s` — causing the NNS to mint more ICP to Swap B's SNS governance treasury than the actual direct participation warrants, draining Neurons' Fund maturity incorrectly.
- A fabricated `sns_governance_canister_id` in the `Committed` variant — redirecting minted ICP to an arbitrary account.
- A premature `Aborted` result — causing the NNS to refund all reserved Neurons' Fund maturity before Swap B's actual swap concludes, permanently preventing Neurons' Fund participation in Swap B.

The function is idempotent only after it has been called once successfully; if Swap B's proposal has not yet been settled, Swap A can race to settle it first with fabricated data.

---

### Likelihood Explanation

Every deployed SNS swap canister is a valid caller. SNS swap canisters are deployed by SNS-W (NNS-controlled) and upgraded via NNS proposals, but once deployed they are independent canisters. A malicious SNS project that passes the NNS governance vote to deploy an SNS would receive a swap canister that satisfies the `is_canister_id_valid_swap_canister_id` check. That canister can then target any other SNS's unsettled Neurons' Fund participation. The attack requires no privileged key, no threshold corruption, and no external dependency — only a deployed SNS swap canister principal.

---

### Recommendation

After the existing `is_canister_id_valid_swap_canister_id` check, add a binding check that the caller is specifically the swap canister recorded in the proposal data for `nns_proposal_id`:

```rust
// After is_canister_id_valid_swap_canister_id passes:
let expected_swap_canister_id = proposal_data
    .swap_canister_id()  // retrieve from proposal's SNS deployment record
    .ok_or_else(|| GovernanceError::new_with_message(
        ErrorType::NotAuthorized,
        "Proposal has no associated swap canister ID".to_string(),
    ))?;
if target_canister_id != expected_swap_canister_id {
    return Err(GovernanceError::new_with_message(
        ErrorType::NotAuthorized,
        format!(
            "Caller {caller} is not the swap canister for proposal {:?}. \
             Expected {expected_swap_canister_id}.",
            request.nns_proposal_id,
        ),
    ));
}
```

```rust
async fn is_canister_id_valid_swap_canister_id(
    target_canister_id: CanisterId,
    env: &dyn Environment,
) -> Result<(), String> {
    let list_deployed_snses_response = env
        .call_canister_method(
            SNS_WASM_CANISTER_ID,
            "list_deployed_snses",
            Encode!(&ListDeployedSnsesRequest {}).expect(""),
        )
        .await
        .map_err(|err| {
            format!(
                "Failed to call the list_deployed_snses method on sns_wasm ({SNS_WASM_CANISTER_ID}): {err:?}",
            )
        })?;

    let list_deployed_snses_response =
        Decode!(&list_deployed_snses_response, ListDeployedSnsesResponse).map_err(|err| {
            format!(
                "Unable to decode response as ListDeployedSnsesResponse: {err}. reply_bytes = {list_deployed_snses_response:#?}",
            )
        })?;

    let is_swap = list_deployed_snses_response
        .instances
        .iter()
        .any(|sns| sns.swap_canister_id == Some(target_canister_id.into()));
    if !is_swap {
        return Err(format!(
            "target_swap_canister_id is not the ID of any swap canister known to sns_wasm: {target_canister_id}"
        ));
    }

    Ok(())
}
```
