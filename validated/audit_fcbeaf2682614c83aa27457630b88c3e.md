### Title
Governance Tally Uses Live Delegation State, Enabling Post-Vote Quorum Manipulation — (`sei-cosmos/x/gov/keeper/tally.go`)

### Summary

Sei's governance `Tally` function reads **current** delegation state at tally time rather than a snapshot taken at vote time. An attacker who holds a large stake can vote YES during the voting period (pushing a proposal over quorum), then submit an `MsgUndelegate` transaction in the same block where the `EndBlocker` processes the proposal. Because `Undelegate` immediately moves tokens from `BondedPool` to `NotBondedPool`, both `totalVotingPower` and `TotalBondedTokens` are reduced at tally time, potentially dropping the proposal below quorum and triggering deposit burns.

### Finding Description

In `Tally`, quorum is checked as:

```go
percentVoting := totalVotingPower.Quo(keeper.sk.TotalBondedTokens(ctx).ToDec())
if percentVoting.LT(quorumThreshold) {
    return false, true, tallyResults   // burnDeposits = true
}
```

`totalVotingPower` is built by iterating **current** delegations of each voter:

```go
keeper.sk.IterateDelegations(ctx, voter, func(..., delegation ...) (stop bool) {
    votingPower := delegation.GetShares().MulInt(val.BondedTokens).Quo(val.DelegatorShares)
    ...
    totalVotingPower = totalVotingPower.Add(votingPower)
})
```

And `TotalBondedTokens` reads the live bonded pool balance:

```go
func (k Keeper) TotalBondedTokens(ctx sdk.Context) sdk.Int {
    bondedPool := k.GetBondedPool(ctx)
    return k.bankKeeper.GetBalance(ctx, bondedPool.GetAddress(), k.BondDenom(ctx)).Amount
}
```

When `Undelegate` is called, the staking module immediately moves tokens from `BondedPool` to `NotBondedPool`:

```go
if validator.IsBonded() {
    k.bondedTokensToNotBonded(ctx, returnAmount)
}
```

Because Cosmos processes transactions before `EndBlocker`, an attacker can submit `MsgUndelegate` in the same block where the voting period expires. The `EndBlocker` then sees zero delegation shares for the attacker and a reduced `TotalBondedTokens`. [1](#0-0) [2](#0-1) [3](#0-2) [4](#0-3) 

### Impact Explanation

**Quorum-failure griefing / deposit burn:**

| Step | State |
|---|---|
| Total bonded = 100, quorum = 33.4% | — |
| Attacker (40%) votes YES; other voters (5%) vote YES | totalVotingPower = 45%, percentVoting = 45% > quorum ✓ |
| Attacker submits `MsgUndelegate` in the same block as `EndBlocker` | BondedPool drops to 60, attacker delegation = 0 |
| Tally runs: totalVotingPower = 5%, TotalBondedTokens = 60 | percentVoting = 8.3% < quorum → `burnDeposits = true` |

All depositors lose their governance deposits. The attacker's tokens enter a 21-day unbonding period (not burned), so the attacker recovers principal at the cost of staking rewards.

**Threshold manipulation (secondary):** An attacker who voted NO can undelegate to remove their NO votes from `totalVotingPower`, flipping a failing proposal to passing. [5](#0-4) [6](#0-5) 

### Likelihood Explanation

- The attack window is narrow: the attacker must submit `MsgUndelegate` in the exact block where `EndBlocker` processes the proposal. Voting end times are public and block times are predictable, making this feasible.
- Requires a large stake (enough to swing quorum). This limits the attacker pool but does not eliminate it.
- Economic cost is only lost staking rewards during the 21-day unbonding period — principal is recovered.
- No governance key or validator key is required; any unprivileged token holder with sufficient stake can execute this.

### Recommendation

1. **Snapshot voting power at vote time.** Record the delegation shares at the moment `AddVote` is called and use those snapshots in `Tally`, rather than querying live delegation state.
2. **Alternatively, snapshot `TotalBondedTokens` at voting start.** Store `VotingStartBondedTokens` on the `Proposal` object when the voting period begins and use that fixed value as the denominator in the quorum check.
3. As a partial mitigation, reject `MsgUndelegate` for addresses that have an active vote on any open proposal (similar to the SPL Governance fix of blocking token withdrawal before finalization). [7](#0-6) [8](#0-7) 

### Proof of Concept

1. Total bonded supply = 100 SEI. Quorum = 33.4%.
2. Attacker holds 40 SEI (40%). Other honest voters hold 5 SEI (5%).
3. Proposal enters voting period. Honest voters vote YES (5%). Attacker votes YES (40%). `percentVoting` = 45% — quorum is met.
4. Attacker monitors the chain. In the block where `BlockTime >= VotingEndTime`, attacker broadcasts `MsgUndelegate` for all 40 SEI. Transaction is included and processed before `EndBlocker`.
5. `EndBlocker` calls `Tally`:
   - `IterateDelegations` for attacker returns 0 shares → contributes 0 to `totalVotingPower`.
   - `TotalBondedTokens` = 60 (attacker's 40 moved to `NotBondedPool`).
   - `totalVotingPower` = 5, `percentVoting` = 5/60 = 8.3% < 33.4% quorum.
   - Returns `burnDeposits = true`.
6. `EndBlocker` calls `keeper.DeleteDeposits`, burning all governance deposits. [9](#0-8) [10](#0-9)

func (keeper Keeper) Tally(ctx sdk.Context, proposal types.Proposal) (passes bool, burnDeposits bool, tallyResults types.TallyResult) {
	results := make(map[types.VoteOption]sdk.Dec)
	results[types.OptionYes] = sdk.ZeroDec()
	results[types.OptionAbstain] = sdk.ZeroDec()
	results[types.OptionNo] = sdk.ZeroDec()
	results[types.OptionNoWithVeto] = sdk.ZeroDec()

	totalVotingPower := sdk.ZeroDec()
	currValidators := make(map[string]types.ValidatorGovInfo)

	// fetch all the bonded validators, insert them into currValidators
	keeper.sk.IterateBondedValidatorsByPower(ctx, func(index int64, validator stakingtypes.ValidatorI) (stop bool) {
		currValidators[validator.GetOperator().String()] = types.NewValidatorGovInfo(
			validator.GetOperator(),
			validator.GetBondedTokens(),
			validator.GetDelegatorShares(),
			sdk.ZeroDec(),
			types.WeightedVoteOptions{},
		)

		return false
	})

func (keeper Keeper) AddVote(ctx sdk.Context, proposalID uint64, voterAddr sdk.AccAddress, options types.WeightedVoteOptions) error {
	proposal, ok := keeper.GetProposal(ctx, proposalID)
	if !ok {
		return sdkerrors.Wrapf(types.ErrUnknownProposal, "%d", proposalID)
	}
	if proposal.Status != types.StatusVotingPeriod {
		return sdkerrors.Wrapf(types.ErrInactiveProposal, "%d", proposalID)
	}

	for _, option := range options {
		if !types.ValidWeightedVoteOption(option) {
			return sdkerrors.Wrap(types.ErrInvalidVote, option.String())
		}
	}

	vote := types.NewVote(proposalID, voterAddr, options)
	keeper.SetVote(ctx, vote)

	// called after a vote on a proposal is cast
	keeper.AfterProposalVote(ctx, proposalID, voterAddr)

	ctx.EventManager().EmitEvent(
		sdk.NewEvent(
			types.EventTypeProposalVote,
			sdk.NewAttribute(types.AttributeKeyOption, options.String()),
			sdk.NewAttribute(types.AttributeKeyProposalID, fmt.Sprintf("%d", proposalID)),
		),
	)

	return nil
}

// Undelegate defines a method for performing an undelegation from a delegate and a validator
func (k msgServer) Undelegate(goCtx context.Context, msg *types.MsgUndelegate) (*types.MsgUndelegateResponse, error) {
	ctx := sdk.UnwrapSDKContext(goCtx)

	addr, err := sdk.ValAddressFromBech32(msg.ValidatorAddress)
	if err != nil {
		return nil, err
	}
	delegatorAddress, err := sdk.AccAddressFromBech32(msg.DelegatorAddress)
	if err != nil {
		return nil, err
	}
	shares, err := k.ValidateUnbondAmount(
		ctx, delegatorAddress, addr, msg.Amount.Amount,
	)
	if err != nil {
		return nil, err
	}

	bondDenom := k.BondDenom(ctx)
	if msg.Amount.Denom != bondDenom {
		return nil, sdkerrors.Wrapf(
			sdkerrors.ErrInvalidRequest, "invalid coin denomination: got %s, expected %s", msg.Amount.Denom, bondDenom,
		)
	}

	completionTime, err := k.Keeper.Undelegate(ctx, delegatorAddress, addr, shares)
	if err != nil {
		return nil, err
	}

	if msg.Amount.Amount.IsInt64() {
		defer func() {
			stakingMetrics.undelegateTotal.Add(goCtx, 1)
			// TODO(PLT-353): remove once staking_undelegate_total verified
			telemetry.IncrCounter(1, types.ModuleName, "undelegate")
			stakingMetrics.undelegateAmount.Record(goCtx, msg.Amount.Amount.Int64())
			// TODO(PLT-353): remove once staking_undelegate_amount verified
			telemetry.SetGaugeWithLabels(
				[]string{"tx", "msg", msg.Type()},
				float32(msg.Amount.Amount.Int64()),
				[]metrics.Label{telemetry.NewLabel("denom", msg.Amount.Denom)},
			)
		}()
	}

	ctx.EventManager().EmitEvents(sdk.Events{
		sdk.NewEvent(
			types.EventTypeUnbond,
			sdk.NewAttribute(types.AttributeKeyValidator, msg.ValidatorAddress),
			sdk.NewAttribute(sdk.AttributeKeyAmount, msg.Amount.String()),
			sdk.NewAttribute(types.AttributeKeyCompletionTime, completionTime.Format(time.RFC3339)),
		),
		sdk.NewEvent(
			sdk.EventTypeMessage,
			sdk.NewAttribute(sdk.AttributeKeyModule, types.AttributeValueCategory),
			sdk.NewAttribute(sdk.AttributeKeySender, msg.DelegatorAddress),
		),
	})

	return &types.MsgUndelegateResponse{
		CompletionTime: completionTime,
	}, nil
}
